# 🏫 Шкільний Telegram Бот

## 📁 Файли проєкту

```
school_bot.py        — головний код бота
school_data.json     — база даних (розклад, ДЗ, юзери, статистика)
requirements.txt     — залежності Python
README.md            — ця інструкція
```

---

## ⚙️ Встановлення

### 1. Встановити Python 3.10+
https://python.org/downloads

### 2. Встановити залежності
```bash
pip install -r requirements.txt
```

### 3. Отримати токен бота
- Написати @BotFather у Telegram
- /newbot → ввести назву → отримати токен

### 4. Вписати токен у school_bot.py
```python
BOT_TOKEN = "ВАШ_ТОКЕН_ТУТ"   # замінити на свій
ADMIN_PASSWORD = "school2025"   # змінити пароль адміна!
```

### 5. Запустити
```bash
python school_bot.py
```

## 🤖 Команди бота

| Команда | Хто | Що робить |
|---------|-----|-----------|
| `/start` | всі | Почати роботу з ботом |
| `/admin ПАРОЛЬ` | адмін | Увійти в панель адміна |

---

## 📱 Для учнів

Після `/start` внизу з'являються 3 кнопки:

- **📚 Розклад** — вибрати день → побачити всі уроки з часом
- **🔔 Дзвінки** — розклад дзвінків з перервами
- **✏️ ДЗ** — вибрати день → побачити домашнє завдання

**Щовечора о 20:00** бот запитує "чи зробив ДЗ?" з кнопками ✅ Так / ❌ Ні.

---

## 🛠 Для адміна

Після `/admin ПАРОЛЬ` відкривається панель:

### Редагувати розклад
1. Натиснути «📚 Редагувати розклад»
2. Вибрати день
3. Надіслати 7 предметів — кожен з нового рядка:
```
Математика
Фізика
Хімія
Біологія
Англійська
Географія
Фізкультура
```

### Додати ДЗ
1. Натиснути «✏️ Додати / редагувати ДЗ»
2. Вибрати день
3. Вибрати урок
4. Написати текст ДЗ
5. Щоб **видалити** ДЗ — надіслати `-`

### Змінити час сповіщення
- Натиснути «🔔 Час сповіщення»
- Ввести час у форматі `20:00`

### Статистика
- Натиснути «📊 Статистика ДЗ»
- Побачити скільки учнів відповіли ✅/❌

## 🚀 Запуск на сервері (щоб бот працював 24/7)

### Варіант 1 — Railway.app (безкоштовно)
1. Зареєструватись на railway.app
2. New Project → Deploy from GitHub
3. Додати змінну середовища `BOT_TOKEN`

### Варіант 2 — systemd (Linux VPS)
```bash
# Створити сервіс
sudo nano /etc/systemd/system/schoolbot.service
```
```ini
[Unit]
Description=School Telegram Bot
After=network.target

[Service]
WorkingDirectory=/home/user/school_bot
ExecStart=/usr/bin/python3 school_bot.py
Restart=always

[Install]
WantedBy=multi-user.target
```
```bash
sudo systemctl enable schoolbot
sudo systemctl start schoolbot
```
